interpolate arg1 t

Full Usage: interpolate arg1 t

Parameters:
    arg0 : Interval<'Units>
    t : float

Returns: Quantity<'Units>

Interpolate between an interval's endpoints based on a parameter value that will generally be between 0.0 and 1.0. A value of 0.0 corresponds to the minimum value of the interval, a value of 0.5 corresponds to its midpoint and a value of 1.0 corresponds to its maximum value: 


     Interval.interpolate (Interval.from 1 5) 0
     --> 1
     Interval.interpolate (Interval.from 1 5) 0.75
     --> 4
Values less than 0.0 or greater than 1.0 can be used to extrapolate: 

     Interval.interpolate (Interval.from 1 5) 1.5
     --> 7
Note that because of how Interval.from works, the interpolation is in fact from the minimum value to the maximum, _not_ "from the first Interval.from argument to the second": 

     Interval.interpolate (Interval.from 0 10) 0.2
     --> 2
     Interval.interpolate (Interval.from 10 0) 0.2
     --> 2 // not 8!

arg0 : Interval<'Units>
t : float
Returns: Quantity<'Units>

interpolationParameter arg1 value

Full Usage: interpolationParameter arg1 value

Parameters:
    arg0 : Interval<'Units>
    value : Quantity<'Units>

Returns: float

Given an interval and a given value, determine the corresponding interpolation parameter (the parameter that you would pass to [interpolate](#interpolate) to get the given value): 


     Interval.interpolationParameter
         (Interval.from 10 15)
         12
     --> 0.4
The result will be between 0 and 1 if (and only if) the given value is within the given interval: 

     Interval.interpolationParameter
         (Interval.from 10 15)
         18
     --> 1.6
     Interval.interpolationParameter
         (Interval.from 10 15)
         9
     --> -0.2
This is the inverse of interpolate; for any non-zero-width interval, 

     Interval.interpolationParameter interval value
         |> Interval.interpolate interval
should be equal to the original value (within numerical round off).

arg0 : Interval<'Units>
value : Quantity<'Units>
Returns: float

reciprocal arg1

Full Usage: reciprocal arg1

Parameters:
    arg0 : Interval<Unitless>

Returns: Interval<Unitless>

Find the inverse of a unitless interval: 


     Interval.reciprocal <|
         Interval.fromEndpoints
             ( Quantity.float 2
             , Quantity.float 3
             )
     --> Interval.fromEndpoints
     -->     ( Quantity.float 0.333
     -->     , Quantity.float 0.500
     -->     )
Avoid using this function whenever possible, since it's very easy to get infinite intervals as a result: 

     Interval.reciprocal <|
         Interval.fromEndpoints
             ( Quantity.float -1
             , Quantity.float 2
             )
     --> Interval.fromEndpoints
     -->     ( Quantity.negativeInfinity
     -->     , Quantity.negativeInfinity
     -->     )
Since zero is contained in the above interval, the range of possible reciprocals ranges from negative to positive infinity!
